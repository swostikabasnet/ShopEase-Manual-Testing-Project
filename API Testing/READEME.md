# API Testing Project (JSON Server)

##  Description
This project demonstrates API testing using JSON Server and Postman.

##  Tools Used
- Postman
- JSON Server

##  Features Tested
- GET all products
- GET product by ID
- POST new product
- PUT update product
- DELETE product
- User API testing(GET)

##  How to Run
1. Install Node.js and JSON Server:
    -npm package(node js)
    -npm install -g json-server(json server)

2. Run:
   json-server --watch shopease.json
3. Use Postman to test endpoints

##  Files Included
- shopease.json (mock database)
- Postman Collection
- API Test Cases
