# ShopEase – Manual Testing Project

## Project Description
This project demonstrates manual testing documentation for a sample e-commerce web application named "ShopEase".

---

##  Modules Covered
- User Registration
- Login
- Add to Cart
- Logout

---

##  Documents Included
The Excel file contains the following sheets:

1. Test Plan & Requirements
3. Test Scenarios
4. Test Cases
5. RTM (Requirement Traceability Matrix)
6. Bug Report

---

##  Tools Used
- Microsoft Excel

---

##  Testing Type
Manual Functional Testing
