# ShopEase – Manual Testing Project

## Project Description
This project demonstrates both Manual Testing and API Testing for a sample e-commerce web application named "ShopEase".

---
## Testing Types Covered
- Manual Functional Testing
- API Testing (using JSON Server & Postman)
  
---
##  Manual Testing - Modules Covered
- User Registration
- Login
- Add to Cart
- Logout

---
##  Documents Included
The Excel file contains the following sheets:

1. Test Plan & Requirements
3. Test Scenarios
4. Test Cases
5. RTM (Requirement Traceability Matrix)
6. Bug Report

---
## API Testing 
API testing is performed using a mock server.

- GET all products
